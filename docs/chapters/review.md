## 🚀 What's Next?

### Create Your Profile README

Want to show off your skills on your GitHub profile? Create a public repository with the same name as your GitHub username and add a `README.md` file — GitHub will display it on your profile page.

**Quick steps:**
1. Create a new public repository named after your GitHub username (e.g., `mona` → repository named `mona`)
2. Create a file named `README.md` in the root of that repository
3. Add some content — introduce yourself, list your skills, link to your projects
4. If you used a branch, open and merge a pull request

📖 Full guide: [Managing your profile README](https://docs.github.com/account-and-profile/setting-up-and-managing-your-github-profile/customizing-your-profile/managing-your-profile-readme)
