# Hands-On Exercises

These four steps walk you through the complete GitHub workflow — from creating a branch to merging your pull request.

| Step | Topic | Time |
|------|-------|------|
| 1 | [Create a Branch](chapter-1-create-a-branch) | ~5 min |
| 2 | [Commit a File](chapter-2-commit-a-file) | ~5 min |
| 3 | [Open a Pull Request](chapter-3-open-a-pull-request) | ~5 min |
| 4 | [Merge Your Pull Request](chapter-4-merge-your-pull-request) | ~5 min |

{: .note }
> To complete these exercises, you'll need a GitHub account. If you don't have one yet, [sign up for free](https://github.com/signup).
