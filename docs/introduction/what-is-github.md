## Overview

**GitHub** is a collaboration platform that uses [Git](https://docs.github.com/en/get-started/quickstart/github-glossary#git) for versioning. It is a popular place to share and contribute to [open-source](https://docs.github.com/en/get-started/quickstart/github-glossary#open-source) software, and it powers millions of projects — from personal experiments to enterprise-grade systems.

## Git vs. GitHub

These two terms are often confused. Here's how they differ:

| | Git | GitHub |
|---|---|---|
| **Type** | Version control system | Cloud-based platform |
| **Purpose** | Track changes in files | Host and collaborate on Git repositories |
| **Where it runs** | Locally on your computer | On the web (github.com) |
| **Created by** | Linus Torvalds (2005) | GitHub, Inc. (2008) |

**Git** is the underlying technology that records changes to files. **GitHub** is the online service that hosts your Git repositories and provides powerful collaboration tools on top of them.

## What Can You Do on GitHub?

GitHub provides a full suite of tools for software development and collaboration:

### 🗂 Store and manage code
Keep all your project files in one place, with a complete history of every change ever made.

### 🌿 Work on branches
Experiment with new features or fixes in isolation, without touching the main project until you're ready.

### 🤝 Collaborate with others
Propose changes via pull requests, review code, leave comments, and discuss improvements — all in one place.

### 🔍 Discover open-source projects
GitHub hosts millions of public repositories. You can explore, fork, and contribute to projects from all over the world.

### ⚙️ Automate workflows
Use [GitHub Actions](https://github.com/features/actions) to build, test, and deploy your code automatically.

## GitHub in the Real World

Here are a few examples of what people build and share on GitHub:

- **VS Code** — Microsoft's popular code editor
- **React** — Facebook's UI library used by millions of websites
- **TensorFlow** — Google's machine learning framework
