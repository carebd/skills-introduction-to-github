## Welcome

People use GitHub to build some of the most advanced technologies in the world. Whether you're visualizing data or building a new game, there's a whole community and set of tools on GitHub that can help you do it even better.

This guide walks you through everything you need to **start contributing in less than an hour**.

---

## What You'll Learn

| Topic | Description |
|-------|-------------|
| 🌿 **Branches** | Create an isolated space to work on changes safely |
| 💾 **Commits** | Record changes to your project with a meaningful message |
| 🔀 **Pull Requests** | Propose, review, and discuss changes before merging |
| ✅ **Merging** | Incorporate your changes into the main project |

---

## Who Is This For?

- 👋 New developers just getting started
- 🐙 New GitHub users exploring the platform
- 🎓 Students learning version control

**Prerequisites:** None — this guide is a great introduction for your first day on GitHub.
